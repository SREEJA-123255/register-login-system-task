# Register Login System

## About
A full stack web application for user registration and login.

## Technologies Used
- HTML, CSS, JavaScript
- Node.js, Express.js
- MongoDB, Mongoose
- bcrypt

## Features
- User Registration
- User Login
- Password Encryption
- Duplicate user check
- Error Handling
- Responsive Design
- Form Validation

## How to Run
1. cd backend
2. npm install
3. Create a .env file in backend folder
4. Add this line in .env file:
   MONGO_URL=your_mongodb_connection_string
5. node server.js
6. Open register.html in browser
